# Project Templates

This folder contains 12 production-ready project templates organized by technology stack.

## Java Templates
- `java/spring-boot-microservice-template` - Spring Boot 3 microservice with Docker and GitLab CI
- `java/rabbitmq-event-driven-architecture` - Event-driven architecture with RabbitMQ
- `java/spring-batch-etl-processor` - Batch processing with Spring Batch

## Node.js Templates
- `nodejs/nestjs-rest-api-template` - NestJS REST API with Prisma and JWT
- `nodejs/express-graphql-server` - GraphQL API with Apollo Server
- `nodejs/nodejs-websocket-realtime` - WebSocket server with Socket.io

## React Templates
- `react/react-admin-dashboard` - Admin dashboard with Recharts
- `react/react-ecommerce-storefront` - E-commerce with cart and checkout
- `react/react-auth-boilerplate` - Authentication with JWT

## Angular Templates
- `angular/angular-admin-panel` - Admin panel with NgRx
- `angular/angular-landing-page` - Landing page with SSR
- `angular/angular-crud-generator` - Dynamic CRUD generator
