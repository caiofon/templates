# Angular CRUD Generator

Dynamic CRUD application with schema-driven forms and tables.

## Features

- ✅ Schema-driven form generation
- ✅ Dynamic table with CRUD operations
- ✅ Generic API service layer
- ✅ Validation rules from schema
- ✅ Export to CSV/Excel
- ✅ Pagination and sorting

## Quick Start

```bash
npm install
ng serve
```
